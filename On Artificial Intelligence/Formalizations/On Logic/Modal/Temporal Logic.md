
Temporal logic models the flow of time. $\Box \phi$ is read as "**Always** $\phi$" (in the future).

### Semantic Interpretation
* **$\Box \phi$**: $\phi$ is true at all future dates.
* **$\Diamond \phi$**: $\phi$ is true at some future date.

### Axiom Behavior
* **Axiom 4 (Transitivity)**: Valid. If something is true for all time, it is true for all time that it is true for all time.
* **Axiom T**: Depends on the model. If the "future" includes the "present" (Reflexive), then T holds. If it is "strictly future" ($w < v$), T fails.
* **Axiom 5 Fails**: Time is generally considered linear or branching forward, not Euclidean.
