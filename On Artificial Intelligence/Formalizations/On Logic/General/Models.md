
A Model (often denoted $m$) is a mathematical abstraction of a "possible world" or a specific configuration of the environment.

### Definition
* A model fixes the truth value of every **Atomic Sentence** in the logic.
* If a sentence $\alpha$ evaluates to True in model $m$, we say:
    * "$m$ satisfies $\alpha$".
    * "$m$ is a model of $\alpha$".

### Notation: $M(\alpha)$
We denote the set of all models where sentence $\alpha$ is true as $M(\alpha)$.
* This notation is critical for defining [[Entailment]].
