Syntax defines the formal rules for constructing well-formed sentences in a logical language.

> [!info] **Definition**
> Syntax answers the question: *"Is this sequence of symbols a valid sentence?"*
> * It does **not** address meaning or truth; it only concerns the arrangement of symbols.
> * Example in Arithmetic: "x + y = 4" is valid syntax. "x4y + =" is invalid syntax.

### Structure
* **Atomic Sentences**: The most basic assertions (e.g., $P$, $Q$).
* **Complex Sentences**: Constructed from atomic sentences using **Logical Connectives** (e.g., $\neg P$, $P \wedge Q$).
