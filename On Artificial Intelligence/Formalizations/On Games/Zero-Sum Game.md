
A special class of constant-sum games where the interests of the players are diametrically opposed.

### Definition
A game is zero-sum if for every strategy profile $s$:
$$\sum_{i=1}^n u_i(s) = 0$$
In a 2-player zero-sum game, this means $u_1(s) = -u_2(s)$.

### Properties
* **Minimax = Maximin**: In zero-sum games, the minimax value and maximin value are equal.
* **Saddle Point**: A pure strategy Nash Equilibrium in a zero-sum game is often called a saddle point.
