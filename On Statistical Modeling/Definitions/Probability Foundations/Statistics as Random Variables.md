## Core Concept: The Unifying Idea
This is the fundamental bridge between Probability (theory) and Statistics (data). 

Data is simply a collection of $n$ observed random variables, denoted as the set $\{x_1, x_2, \dots, x_n\}$. These observations are generated by the abstract random variants $X_1, \dots, X_n$.

When we collect a massive amount of data, we need a way to summarize it. We do this by defining a **Statistic** (often denoted as $T$). 

## The Mathematical Definition of a Statistic
A statistic is strictly a function evaluated over the sample data. It takes a dataset of $n$ observations and compresses it into a single number.
$$T = g(X_1, X_2, \dots, X_n)$$
*Examples:* The sample mean ($\bar{X}$) and the sample variance ($S^2$) are statistics. 


## The Critical Implication
Because a statistic $T$ is computed by applying a function to random variables ($X_1 \dots X_n$), **a function of random variables is itself a random variable.**

Therefore, a statistic is random. 

If you run the experiment again and collect a *different* dataset, the function will output a *different* statistic. Because statistics are random variables, they possess their own [[Probability Distributions (PDF, PMF, CDF)]] (often called sampling distributions).
