## Core Concept
A [[Probability Distributions (PDF, PMF, CDF)|Probability Distribution]] is not just a descriptive curve; it is the theoretical engine of reality, formally known as the **Data Generating Process**.

## The Estimation Problem
We are trapped behind an observational wall. We *always* only see the empirical data generated by the distribution (the outputs), but we never see the true mathematical distribution itself. 

Knowing the exact, true probability distribution would allow us to derive every possible information point of interest with perfect certainty. However, these true distributions are **fundamentally unknowable**.


Because the true engine is hidden, our goal as statisticians is to **estimate** the distribution using the data it spit out. 
* **Non-parametric Estimation:** Trying to guess the shape visually (e.g., using binning or histograms) based on the frequency of the observed data.
* **Parametric Estimation:** Imposing a pre-defined mathematical structure onto the data to simplify the problem. See [[Defining Parametric Families]].
