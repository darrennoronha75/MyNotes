# Gaussian Linear Model

## Definition
The Gaussian Linear Model establishes an explicit, stochastic relationship between variables. Instead of a purely deterministic geometric line ($y = mx + b$), it models a response variable ($Y$) as a linear combination of predictors ($X$) plus an uncontrollable, random noise component ($\epsilon$).

$$Y = \beta_0 + \beta_1X + \epsilon$$

* **$\beta_0$ (Intercept):** The expected value of $Y$ when all $X$ are zero.
* **$\beta_1$ (Slope Coefficient):** The expected change in $Y$ for a one-unit change in $X$.
* **$\epsilon$ (Error/Residual):** The unobservable stochastic noise generated by the [[Data Generating Process]].

## Strict Assumptions
For the mathematical mechanics of the model to hold valid (specifically, for hypothesis tests on the coefficients to work), the error term ($\epsilon$) must adhere to four strict structural rules:
1. **Expected Value of Zero:** The noise balances out on average ($E[\epsilon] = 0$).
2. **Homoscedasticity:** The variance of the noise is constant across all values of $X$ ($Var(\epsilon) = \sigma^2$).
3. **Normality:** The errors are distributed according to the [[Normal Distribution]].
4. **Independence:** Knowing the error of one observation provides zero information about the error of another observation.

**Connections:**
* Because the true parameters ($\beta$ and $\sigma^2$) are "nuisance parameters" that we cannot see, we must estimate them using [[Ordinary Least Squares (OLS)]].
