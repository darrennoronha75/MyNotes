# Coefficient of Determination (R-squared)

🚧 **Note: Stub file to be expanded.**
## Definition
A statistical measure that represents the proportion of the variance for a dependent variable that is mathematically explained by an independent variable (or variables) in a [[Gaussian Linear Model]].
