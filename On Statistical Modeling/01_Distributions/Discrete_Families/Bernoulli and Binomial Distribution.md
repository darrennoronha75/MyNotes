# Bernoulli and Binomial Distributions

🚧 **Note: Stub file to be expanded.**
## Definition
* **The Bernoulli:** Models a *single* binary event (1 or 0, success or failure). Parameter: $p$ (probability of success).
* **The Binomial:** Models the number of successes in multiple, independent Bernoulli trials.

---
**Connections:**
* Forms the foundation for the logistic link function in Logistic Regression.
