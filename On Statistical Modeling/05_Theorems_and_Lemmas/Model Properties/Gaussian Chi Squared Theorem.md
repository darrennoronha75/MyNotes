# Gaussian Chi Squared Theorem

🚧 **Note: This is a stub file to be expanded when processing the advanced PDF corpus.** ## Definition
Consider a Gaussian linear model. 
a) The random vectors $\hat{\gamma}$ and $\epsilon$ are independent and satisfy $\hat{\gamma} \sim Normal(\gamma, \sigma^2(A^TA)^{-1})$ and $\epsilon \sim Normal(0, \sigma^2(I_n - A(A^TA)^{-1}A^T))$. 
b) If $n > k+1$, one has $\|\epsilon\|^2/\sigma^2 \sim \chi^2_{n-k-1}$.
