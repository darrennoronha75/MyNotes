# Gaussian MLE Theorem

🚧 **Note: This is a stub file to be expanded when processing the advanced PDF corpus.** ## Definition
Consider a Gaussian linear model and let $\sigma^2_{ML} = \frac{1}{n} \|\epsilon\|^2$. Then, $(\hat{\gamma}, \sigma^2_{ML})$ is the maximum likelihood estimator for $(\gamma, \sigma^2)$.
