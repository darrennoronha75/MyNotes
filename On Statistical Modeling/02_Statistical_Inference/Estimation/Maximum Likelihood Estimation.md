# Maximum Likelihood Estimation (MLE)

🚧 **Note: Stub file to be expanded with PDF Chapter 3 content.**
## Definition
A method of estimating the parameters of an assumed [[Data Generating Process]] by finding the parameter values that maximize the likelihood function (making the observed data most probable).

---
**Connections:**
* Contrasts with [[Ordinary Least Squares (OLS)]].
